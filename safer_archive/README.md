# SAFER Archive

This folder contains archived copies of source data and reports used in the analysis, 
provided to ensure long-term accessibility for reproducibility.

## Contents

- `SAFER_RA.csv` — SAFER Risk Assessment dataset (State Water Resources Control Board, accessed August 2025)
- `2025needsassessment.pdf` — 2025 Drinking Water Needs Assessment report (SWRCB, 2025)

## Original Sources

- SWRCB Needs Assessment: https://www.waterboards.ca.gov/drinking_water/certlic/drinkingwater/needs.html
- California Open Data Portal: https://data.ca.gov/dataset/safer-failing-and-at-risk-drinking-water-systems
