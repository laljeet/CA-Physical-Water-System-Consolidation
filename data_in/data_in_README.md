# Input Data

This directory contains input datasets required for the analysis. Due to file size, these files are not tracked in the repository and must be downloaded separately.

## Required Files

### 1. SAFER Risk Assessment (SAFER_RA.csv)
- **Source:** State Water Resources Control Board
- **URL:** https://www.waterboards.ca.gov/drinking_water/certlic/drinkingwater/needs.html
- **Description:** System-level risk classifications, service connections, population, and geographic coordinates
- **Place at:** `data_in/SAFER_RA.csv`

### 2. California Drinking Water System Area Boundaries
- **Source:** State Water Resources Control Board
- **URL:** https://gis.data.ca.gov/datasets/CDPH::california-drinking-water-system-area-boundaries
- **Description:** Service area boundary polygons for California public water systems
- **Place at:** `data_in/California_Drinking_Water_System_Area_Boundaries/` (extract full shapefile here)

### 3. OSM Road Networks
- **Source:** OpenStreetMap via Overpass API
- **Description:** Road network data for routing analysis
- **Generated by:** Running `R/04a_download_osm_roads.R`
- **Stored at:** `data_in/osm_roads/<county_slug>_roads_osm.rds`
- **Note:** Download requires internet access and may take 30-60 minutes

### 4. County Boundaries
- **Source:** U.S. Census Bureau TIGER/Line
- **Description:** California county boundaries
- **Generated by:** Running `R/01_get_counties.R` (downloads automatically via `tigris` package)
- **Stored at:** `data_in/counties_ca.gpkg`
